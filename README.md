# CineSnatch v4

A lightweight, high-performance media downloader built with Python and CustomTkinter. CineSnatch allows users to search, manage, and download media content with seamless integration across multiple debrid services.

## 🚀 Status: Migration in Progress
I am currently migrating the backend logic from **AllDebrid** to the **TorBox API**. This update will improve download speeds and provide better support for multi-service cached files.

## ✨ Features
* **Modern UI:** Built with CustomTkinter for a sleek, dark-themed experience.
* **Smart Search:** Integrated with TMDB and OMDB for accurate movie and show metadata.
* **Subtitle Engine:** Built-in support for OpenSubtitles and Subdl to find the best matches automatically.
* **Debrid Integration:** Real-time checking of cached hashes to ensure instant downloads.

## 🛠️ Tech Stack
* **Language:** Python 3.10+
* **GUI:** CustomTkinter
* **APIs:** TorBox, AllDebrid, TMDB, OpenSubtitles

## 🔧 Installation & Usage
1. Clone the repository:
   `git clone https://github.com/xpl01tz/torbox-downloader-project.git`
2. Install dependencies:
   `pip install -r requirements.txt` (Coming soon)
3. Run the application:
   `python cinesnatch.py`
